# 每行代表一个抓包任务
# 每行参数分别为：抓包IP（可为any），抓包端口（可为any），网卡IP（或网卡名称，可为any），每多少秒一个切片，共存储多少切片（新的会覆盖旧的）
# 示例：
# 224.5.6.7 23456 192.168.0.100 60 10
# 抓包IP：224.5.6.7，抓包端口：23456，网卡IP：192.168.0.100，每60秒一个切片，存10片（第11片会覆盖第1片）
#
# 192.168.0.1 any 127.0.0.1 30 100
# 抓包IP：192.168.0.1，抓包端口：任意，网卡IP：127.0.0.1，每30秒一个切片，存100片（第101片会覆盖第1片）
#
# any 12345 enp5s0 120 30
# 抓包IP：任意，抓包端口：12345，网卡名称：enp5s0，每120秒一个切片，存30片（第31片会覆盖第1片）
